Browser Compatibility

Tested in:

  • Internet Explorer 6
  • Internet Explorer 7
  • Internet Explorer 8
  • Firefox 3.5
  • Safari 4
  • Google Chrome 3
  • Opera 9
  • Opera 10

Limitations

Firefox does not currently support the CSS 3 property "text-overflow" so a XUL binding is used to achieve the desired effect in Firefox, introducing the following limitations in that browser:

  • Truncated text can not be selected.
  • Any whitespace before the start of your text will be represented, introducing a unremovable indentation.
  • Child nodes can only be nested one level deep and must also have the XUL binding applied.
